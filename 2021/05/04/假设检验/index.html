假设检验

发表于2021-05-04|更新于2021-05-09
|字数总计:1.4k|阅读时长:5分钟

Hypothesis Testing

根据样本信息来检验关于总体的某个假设是否正确，可分为两类：

  • 参数检验问题：总体分布已知，检验某未知参数的假设
  • 非参数检验问题：总体分布未知时的假设检验问题

假设检验的方法

先假设所做的假设 $H_0$ 成立，然后从总体中取样，根据样本的取值来判断是否有‘不合理’的现象出现，最后做出接受或者拒绝所做假设的决定‘不合理’的现象指小概率事件在一次事件中几乎不会发生。

在假设检验中，需要对‘不合理’的小事件给出一个定性描述，通常给出一个上界$\alpha$，当一事件发生的概率小于$\alpha$时则成为小概率事件，通常取$\alpha = 0.05, 0.1, 0.01$，其具体取值根据实际问题而定。在假定$H_0$成立下，根据样本提供的信息判断出不合理的现象，则认为假设$H_0$不显著，$\alpha$被称为显著水平。

不否定假设$H_0$并不是肯定假设$H_0$一定成立，而只能说差异不够显著，没达到否定的程度，所以假设检验被称为“显著性检验”。

假设检验的步骤

（1）根据实际问题提出原假设$H_0$和备择假设$H_1$

（2）确定检验统计量（分布已知）

（3）确定显著性水平$\alpha$，并给出拒绝域

（4）由样本计算统计量的实测值，判断是否接受原假设$H_0$

  • 原假设$H_0: \mu = \mu_0$和备择假设$H_1:\mu \not= \mu_0$，双边假设检验

  • 原假设$H_0:\mu \leq \mu_0$和备择假设$H_1:\mu > \mu_0$，右边检验

  • 原假设$H_0:\mu \geq \mu_0$和备择假设$H_1:\mu<\mu_0$，左边检验

两类错误

  • 第一类错误：当$H_0$为真时，我们仍可能拒绝$H_0$

  • 第二类错误：当$H_0$不成立时，我们仍可能接受$H_0$

犯第一类错误的概率为$\alpha$，即显著性水平

第二类错误的概率用$\beta$表示

  • Pearson原则：在控制第一类错误的前提下，尽可能减小第二类错位的概率

正态总体期望的假设检验

  • $\mathbf{Z}$检验（方差已知的单个正态总体的期望检验）

设$X_1, X_2, \cdots, X_n$是来自总体$X\sim N(\mu, \sigma^2)$的样本，若方差$\sigma^2$已知，检验原假设$H_0:\mu = \mu_0$和备择假设$H_1:\mu \not= \mu_0$，设样本均值为$\bar{X} = \sum_{i=1}^{n}X_i/n$，根据正态分布的性质选择检验统计量

给定显著性水平$\alpha$，得到拒绝域为$|Z| \geq \mu_{\alpha/2}$，这种检验方法称为$\mathbf{Z}$检验法

  • $\mathbf{t}$检验（方差未知的单个正态总体的期望检验）

设$X_1, X_2, \cdots, X_n$是来自总体$X\sim N(\mu, \sigma^2)$的样本，若方差$\sigma^2$未知，检验原假设$H_0:\mu = \mu_0$和备择假设$H_1:\mu \not= \mu_0$，设样本均值为$\bar{X} = \sum_{i=1}^{n}X_i/n$和样本修正方差$S^2 = \sum_{i=1}^{n}(X_i-\bar{X})^2/(n-1)$,根据正态分布的性质选择检验统计量

给定显著性水平$\alpha$，得到拒绝域为$|t|\geq t_{\alpha/2}(n-1)$，这种检验方法称为$\mathbf{t}$检验法

设$X_1, X_2, \cdots, X_n$是来自总体$X\sim N(\mu, \sigma_1^2)$的样本，以及$Y_1, Y_2, \cdots, Y_m$是来自总体$Y\sim N(\mu_2, \sigma_2^2)$的样本，若方差$\sigma_1^2,\sigma_2^2$已知，检验原假设$H_0:\mu_1-\mu_2=\sigma$和备择假设$H_1:\mu_1-\mu_2\not=\delta$（$\delta$为常数）。设样本均值$\bar{X} = \sum_{i=1}^{n}X_i/n$和$\bar{Y} = \sum_{i=1}^{m}Y_i/m$，根据正态分布的性质有

正态分布的方差假设检验

  • $\chi^2$检验（单个正态总体的方差检验）

设$X_1, X_2, \cdots, X_n$是来自总体$X\sim N(\mu, \sigma^2)$的样本，检验原假设$H_0:\sigma^2=\sigma_0^2$和备择假设$H_1:\sigma^2\not= \sigma_0^2$。设样本修正方差$S^2 = \sum_{i=1}^{n}(X_i-\bar{X})^2/(n-1)$，根据正态总体抽样定理选择检验统计量
