强化学习的概念

Created2021-05-05|Updated2021-05-09
|Word count:726|Reading time:2min

强化学习

强化学习（reinforcement learning）是指智能系统在与环境的连续互动中学习最优行为策略的机器学习问题。假设智能系统与环境的互动基于马尔可夫决策过程（Markov decision process），智能系统能观测到的是与环境互动得到的数据序列。强化学习的本质是学习最优的序贯决策。

智能系统与环境的互动如图所示，在每一步$t$，系统从环境观测状态$s_t$，与奖励$r_t$，并采取一个动作$a_t$。根据系统进行的动作，可以得到下一步$t+1$的状态$s_{t+1}$和奖励$r_{t+1}$。要学习的策略表示为给定的状态下采取的动作，智能系统的目标不是短期奖励的最大化，而是长期累积奖励最大化。强化学习过程中，系统不断地试错（trial and error），以达到学习最优策略的目的。

马尔可夫决策过程

马尔可夫决策过程是状态、奖励、动作序列上的随机过程，用五元数$\langle S, A, P, r, \gamma \rangle$表示

  • $S$是有限状态（state）的集合

  • $A$是有限动作（action）的集合

  • $P$是状态转移概率（transition probability）函数：

  • $r$是奖励函数（reward function）：$r(s,a)=E(r_{t+1}\mid s_t=s, a_t = a)$

  • $\gamma$是衰减系数（discount factor）：$\gamma \in [0,1]$

下一状态只依赖于前一个状态与动作，由状态转移概率函数$P(s’\mid s, a)$表示。下一个奖励依赖于前一个状态与动作，由奖励函数$r(s,a)$表示

一些概念

策略$\pi$定义为给定状态下动作的函数$a=f(s)$或者条件概率分布$P(a\mid s)$。给定一个策略$\pi$，智能系统与环境互动的行为就已确定（并不代表智能系统的行为是确定性的，只是说采用哪种行为是确定的）

价值函数（value function）或状态价值函数（state value function）定义为策略$\pi$从某一状态$s$开始的长期累积奖励的数学期望：

动作价值函数（action value function）定义为策略$\pi$从某一状态$s$和动作$a$开始的长期累积奖励的数学期望：

强化学习的目标就是在所有可能的策略中选出价值函数最大的策略$\pi^\star$，而在实际学习中往往从具体的策略出发，不断优化已有策略。这里$\gamma$表示未来的奖励会有衰减
